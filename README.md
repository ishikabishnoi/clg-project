# DoveTail 
A student-built web platform that brings notes, lectures, PYQs 
and a study planner together in one place. No more hunting through 
drives and group chats every semester.

**Live:** https://mydovetail.netlify.app

## Features
- Secure login and signup
- Personal dashboard with deadline tracker
- Syllabus viewer and week-wise study planner
- Upload and access subject-wise notes
- Save YouTube lecture playlists by subject
- Share and access previous year question papers
- Bookmark resources to your personal study vault

## Built With
- HTML, CSS, Vanilla JavaScript
- Supabase — database, auth and file storage
- Netlify — deployment
